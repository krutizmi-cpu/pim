# PIM — каталог 1.1

Это текущая рабочая версия каталога товаров для торговой компании.

## Что уже есть

- SQLite база товаров `data/catalog.db`
- импорт каталога из Excel
- защита от загрузки пустых и служебных строк
- шаблон Excel для импорта, который можно скачать прямо из интерфейса
- каталог с поиском и фильтрами
- карточка товара с ручным редактированием
- экспорт текущей выборки в Excel
- заготовка для обогащения данных

## Какие поля поддерживаются

Обязательные:

- `article`
- `name`

Необязательные:

- `barcode`
- `weight` (в кг)
- `length`, `width`, `height` (в см)
- `supplier_url`
- `image_url`
- `description`

## Как работать

1. Запусти приложение:
   `streamlit run app.py`
2. В блоке **Загрузка каталога из Excel** скачай шаблон.
3. Заполняй данные на листе `catalog`.
4. Загрузи файл обратно в приложение.
5. Открой карточку товара и при необходимости дозаполни поля вручную.
